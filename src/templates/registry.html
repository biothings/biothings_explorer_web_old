Introduction

Each bio-entity in BioThings Explorer is assigned with a unique identifier. For example, ncbigene ID has a unique identifier, which is "http://identifiers.org/ncbigene". In addition, each URI has a corresponding short form identifier, also called prefix. For example, the prefix for ncbigene ID is ncbigene.
